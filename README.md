BajaCore

A lightweight, barebones Pygame starter template designed to get projects up and running instantly.

🚀 Overview

This is my personal Pygame quickstart template for avoiding repetitive setup when starting new projects. It provides a clean structure, a working window, and a basic game loop so I can jump straight into development.

Instead of rewriting boilerplate every time, I can clone this and start building immediately.

✨ Features
* Pre-configured Pygame window
* Basic game loop (update/render structure)
* Example sprite loading to confirm everything works
* Simple, clean project structure
* Ready to use asset folders
* Minimal and easy to expand
